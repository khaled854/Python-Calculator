Calculator 5 arithmetic operations with a single function

A Python calculator that takes two numbers and an operation from the user, passes them to a calc() function, and prints the result. Handles +, -, *, /, and %.

How It Works
01
ƒ

calc() function

Receives num1, num2, and operation as parameters. Returns the computed result, or an error string if the operation isn't recognised.

02
🔀

if / elif / else

Each of the five operations (+, -, *, /, %) has its own elif branch. The else branch catches anything not in that list and returns an error message.

03
🔢

float() input

Both numbers are read from the user and converted with float(), so the program accepts whole numbers and decimals alike.

04
📤

print result

The return value of calc() is stored in a variable and printed with a simple print('result is', result) statement.
